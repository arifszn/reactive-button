Theming

Reactive Button provides flexible styling option. Most of the common style changes can be achieved by using the style prop.
<ReactiveButton
    style={
        {
            borderRadius: '10px', 
            color: 'black'
        }
    }
/>

However, you are free to create your own theme.

Custom Theme#
